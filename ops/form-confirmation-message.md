# Googleフォーム 確認メッセージ（送信直後にブラウザに表示）

Google フォームの `設定 → プレゼンテーション → 確認メッセージ` に下記本文を貼り付ける。
フォーム送信者へのお礼表示として機能するため、`ops/email-templates.md` の T1（自動返信）の役割を補う。

---

見積もりまでは無料ですので、ご検討段階のご相談も歓迎します。

## 設置後のテスト

1. フォームのプレビューを開き、テスト回答を送信
2. 送信完了画面にこのメッセージが表示されることを確認
3. 回答用スプレッドシートに記録されていることを確認

## 「回答のコピーを回答者に送信」も合わせて設定

- `設定 → 回答 → メールアドレスを収集する` をオン
- `回答のコピーを回答者に送信` を `常に表示` または `リクエストされた場合` へ
- 回答者は自分の控えメールを受け取れる（本サービス側からの追加自動返信は Gmail 下書き機能などで別途運用）

## 注意

- `contact@example.com` は仮置き。`config/site-values.json` の `contact_email` を変更して `npm run site:apply` を実行すれば、本ファイルと法務ページ・LP・テンプレ類の連絡先が `config/.site-values.lock.json` を経由して一括追従する。
