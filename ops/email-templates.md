# 返信メールテンプレ

問い合わせ受領から納品後フォローまで、各段階の返信テンプレ。`{}` は差し替え項目。

---

## T1. 問い合わせ受領の自動返信（Googleフォームで即時）

**件名：** ご送信ありがとうございます / フォーム仕分け自動化パック

```
{お名前} 様

この度はお問い合わせいただきありがとうございます。
「フォーム仕分け自動化パック」です。

内容を確認のうえ、24時間以内（土日祝を除く）に折り返しご連絡いたします。
お急ぎの場合はこのメールへ直接ご返信ください。

----
フォーム仕分け自動化パック
LP: {LP URL}
返信目安: 24時間以内
```

## T2. ヒアリング返信（24時間以内）

**件名：** Re: ご送信ありがとうございます / ヒアリング事項

```
{お名前} 様

お問い合わせありがとうございます。フォーム仕分け自動化パックの {担当者名} です。

いただいた内容を拝見しました。プランご提案にあたって、いくつか確認させてください。

1. 回答シートの現在の列構成
   （差し支えなければ、ヘッダ行だけのスクリーンショットでも構いません）
2. 仕分けに使いたい判定基準
   例：選択肢の値 / 金額の大小 / メールドメイン / キーワード
3. 通知が必要な場合、どなたに届けたいか
   例：「採用」は人事宛、「見積もり」は営業宛 など
4. 希望納期

ご返信いただき次第、最適なプランと作業範囲、確定金額をお送りします。
この時点までは費用は発生しません。

----
フォーム仕分け自動化パック / {担当者名}
```

## T5. 納品のご連絡

**件名：** 納品のご連絡 / フォーム仕分け自動化パック

```
{お名前} 様

本日、作業が完了しましたのでご連絡いたします。

■ 納品物
- スプレッドシートへのスクリプト組み込み（完了）
- 設定シート（_config / _rules / _log）
- 運用マニュアルPDF（添付）

- データと権限の取り扱いPDF（添付）
- テスト送信結果のスクリーンショット（添付）

■ 動作確認のお願い（7日以内）
以下の手順でご確認ください。
1. 実際のフォームから1件送信
2. 該当カテゴリのシートに行が追加されていることを確認
3. 通知メールが届くことを確認（該当プラン）

7日以内に動作確認のご連絡をいただけない、または動作不良の場合は
全額返金いたします。

■ サポート期間
{YYYY-MM-DD} までの {30 or 60} 日間、軽微な修正は無償で承ります。

編集権限はこちらでは今後使用しませんので、ご都合のよいタイミングで
削除してください（手順書に記載）。

ご不明点があれば、いつでもご返信ください。

----
フォーム仕分け自動化パック / {担当者名}
```

## 運用メモ

- 各テンプレは `{担当者名}` `{屋号}` 等を事前に一括置換してから送る
- 初回受注までは手動で送信し、慣れてきたらGmailの「テンプレート」機能に登録
- 辞退返信（T8）は必ず1回で完結させる。追撃しない
