# 営業先リスト作成ガイド

「誰に」「どこから探して」「どうアプローチするか」の3点をまとめた運用ガイド。

## 探し先マップ

### 1. 大学サークル・学生団体（X）

**探し方**

- 検索ワード：「〇〇大学 サークル」「新歓 2026」「イベント運営 学生」
- プロフィールに「Googleフォーム」「申込」「参加受付」と書いてあるアカウントを狙う
- 新歓シーズン（3〜5月）・学祭シーズン（9〜11月）が最も刺さる

**アプローチ**

- 新歓/イベントの申込を告知している投稿にリプライでなく、DMを使う（公開リプはサークル内での安っぽさに繋がる）
- 文面：`sales/outreach-x-dm.md` のカジュアル版
- 価格訴求はライト（14,800円）

### 2. イベント運営（X / PR TIMES / Peatix / connpass）

**探し方**

- Peatix・connpass の主催者一覧
- PR TIMES で「イベント 申込 フォーム」「セミナー 開催」を検索
- X で「申込フォームはこちら」を投稿しているアカウント

**アプローチ**

- 直近のイベント規模（申込件数）を想定して、件数多めの主催者を優先
- 文面：`sales/outreach-email.md` のフォーマル版
- スタンダード（29,800円）訴求

### 3. 小規模事業者・個人事業主（X / Instagram / ホームページ）

**探し方**

- 検索：「お問い合わせフォーム 業種名」
- 自社サイトに Google フォームを埋め込んでいる事業者
- 「自社サイトの問い合わせから届いた申込を社内共有している」と読める投稿

**アプローチ**

- ビジネスアカウントはDMを閉じていることも多い。問い合わせフォームから連絡
- 文面：`sales/outreach-email.md`
- スタンダード or カスタム

### 4. 研究室・大学教員（大学HP / 研究者データベース）

**探し方**

- 大学の研究室HP、researchmap の研究者検索
- アンケート調査・シンポジウム運営をしている研究室を優先

**アプローチ**

- 大学メール宛にフォーマルなメール（T2テンプレの調整版）
- 返信率は低いが、決まれば継続案件化しやすい

### 5. 商工会・自治体・地域団体

**探し方**

- 商工会の会員名簿（公開されている範囲）
- 地方自治体の補助金申請窓口（Googleフォーム利用が多い）

**アプローチ**

- 電話は避ける。必ずメールで。
- カスタム前提で設計

## リストの管理項目

Googleスプレッドシートで管理。最低限これだけ。

| 列 | 項目 |
|---|---|
| A | 優先度（A/B/C） |
| B | 種別（サークル/イベント/事業者/研究室/団体） |
| C | 名称 |
| D | X アカウント or サイトURL |
| E | 接触経路（X DM / メール / 問い合わせフォーム） |
| F | 接触日 |
| G | ステータス（未／送信済／返信あり／辞退） |
| H | メモ（相手の具体的状況） |

## アプローチの基本姿勢

- **1通1カスタム**：1文目に「相手の状況を観察した一文」を必ず入れる
- **売り込み色は薄く**：「合いそうか見てもらえるだけで嬉しい」のトーン
- **価格と返金保証を初回で開示**：匿名運営の警戒は金銭情報で相殺
- **追撃しない**：返信なければ1回でクローズ
- **相手の投稿を数秒でも読む**：テンプレ丸投げはほぼバレる

## 1日のルーティン（営業日）

- 朝：前日の返信対応（15分）
- 昼：新規DM/メール 5〜10件（30分）
- 夕：X投稿1件 or リプ2件（10分）
- 夜：問い合わせ対応（都度）

**1日1時間・週6で30日続ければ、リスト200件は必ず作れる。**
