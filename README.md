# GAS 自動化ポートフォリオ① — フォーム回答自動振り分けツール

## 概要

Googleフォームの回答をスプレッドシートに自動集計し、
指定した列の値（カテゴリ）ごとにシートを分けて整理するツールです。

**よくあるユースケース**
- アンケートの回答を部署・地域・評価ごとに自動仕分け
- イベント申込を日程・参加形態ごとにシート分割
- 問い合わせフォームをカテゴリ別に自動整理

## 機能

- フォーム回答の自動取得
- 指定列の値による条件分岐・シート振り分け
- シートが存在しなければ自動作成
- ヘッダー行の自動フォーマット（太字・色付き・固定）
- フォーム送信トリガーによるリアルタイム自動振り分け
- 手動での一括振り分けにも対応

## 使用技術

- Google Apps Script
- Google Spreadsheet
- Google Forms

## セットアップ手順

### 1. フォームとスプレッドシートを準備
1. Googleフォームを作成する
2. フォームの回答先としてスプレッドシートを連携する

### 2. GASを設定
1. スプレッドシートのメニュー「拡張機能 → Apps Script」を開く
2. `Code.gs` の内容をそのまま貼り付ける
3. `CONFIG` の値を自分の環境に合わせて変更する
   - `SOURCE_SHEET_NAME`: フォーム回答が入るシート名（デフォルト：`フォームの回答 1`）
   - `CATEGORY_COLUMN`: 振り分けに使う列番号（A列=1, B列=2, C列=3...）

### 3. 初回実行
1. `sortAllResponses` を選択して実行（既存データの一括振り分け）
2. 権限の承認ダイアログが出るので許可する

### 4. 自動実行の設定
1. `setAutoTrigger` を実行する
2. 以降、フォームが送信されるたびに自動で振り分けされる

## ファイル構成

```
Code.gs
├── CONFIG           … 設定値（シート名、列番号）
├── sortAllResponses … 全データ一括振り分け
├── onFormSubmit     … フォーム送信時の自動振り分け
├── getOrCreateSheet … シート取得 or 新規作成
├── formatHeader     … ヘッダー行の書式設定
├── setAutoTrigger   … トリガー自動設定
└── removeAutoTrigger… トリガー削除
```

## カスタマイズ例

- `CATEGORY_COLUMN` を変えるだけで、どの列でも振り分け可能
- `formatHeader()` を編集すれば、シートの見た目を変更可能
- `onFormSubmit()` に通知処理を追加すれば、Slack/メール連携も可能

## ポイント

- 手作業でのコピペ・整理作業を完全自動化
- 人為的なミスを防止
- リアルタイム反映で常に最新状態
- コード変更なしで設定値だけで運用可能
