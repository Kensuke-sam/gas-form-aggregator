# このLPで不足している素材

1. **動作デモ動画（30〜60秒）**
   テスト用フォームへ送信 → スプレッドシートの該当カテゴリに自動で行が追加される様子。匿名運営の信用不足を最も強く補える。ファーストビュー直下に置くと離脱率が下がる。
   → 撮影台本を `sales/demo-shotlist.md` に整備済み。

2. **Before/Afterのスクリーンショット**
   「1枚のシートに全部混ざった状態」→「カテゴリ別シートに自動で並んだ状態」の比較画像。文章より圧倒的に早く価値が伝わる。
   → デモデータ投入〜撮影〜LP 埋め込みの手順を `sales/demo-shotlist.md` に整備済み。配置先は `docs/assets/`。

3. **OGP画像（1200×630）**
   SNS でシェアされた際のアイキャッチ。`docs/assets/og-image.png` として配置。作り方は `docs/assets/README.md`。

4. **favicon.ico（任意・レガシーブラウザ用）**
   `favicon.svg` は配置済み。古いブラウザで表示させたい場合のみ `favicon.ico` を追加。

5. **初回導入者向けの特別条件（期間限定）**
   「最初の3件限定：ライトを9,800円／スタンダードに含まれる通知メール設定を無料追加」など。受注ゼロの状態を抜けるための初期ブースター。LP本体に常設せず、期間限定バナーで出すのが望ましい。

## 優先度

1. 動作デモ動画 → 最優先。これがないと匿名運営では信用が積み上がらない
2. Before/Afterスクリーンショット → LP着地5秒以内の離脱を減らす
3. OGP画像 → SNS でのシェア時に必要
4. 初期ブースター → 初受注のハードルを下げるための即効策
