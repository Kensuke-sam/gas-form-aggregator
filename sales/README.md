# 販売資料インデックス

このディレクトリは「フォーム仕分け自動化パック」を販売するための資料一式。
LP、営業文面、価格、FAQ、NG表現などをここから差し替えるだけで運用できる状態にしている。

## ファイル一覧

| ファイル | 用途 |
|---|---|
| `positioning.md` | 誰の何を解決するか／想定読者／悩み |
| `LP.md` | 1ページ完結型LPの最終原稿 |
| `pricing.md` | 3プランの価格表 |
| `faq.md` | FAQ |
| `cta.md` | CTA文言候補 |
| `inquiry-form.md` | 問い合わせフォームで聞くべき項目 |
| `outreach-x-dm.md` | X（旧Twitter）のDM営業文 |
| `outreach-email.md` | メール営業文 |
| `trust-boosters.md` | 匿名運営でも信用を上げる要素 |
| `copy-rules.md` | LPで言わない方がいい表現 |
| `feature-benefit.md` | 機能 → 顧客価値の変換表 |
| `missing-assets.md` | このLPで不足している素材 |

## 運用方針

- 売り方は「ツール販売」ではなく「Googleフォーム整理の導入サービス」
- 価格はまず記載の3プラン。売れ行きを見て調整
- 初受注までは全額返金保証を前面に出す
- 匿名運営の不安は「作業手順の全工程開示」「編集権限の一時付与＆返却」でカバー
