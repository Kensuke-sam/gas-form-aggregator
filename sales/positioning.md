# ポジショニング

## ひとことで何を売るサービスか

Googleフォームの回答を、カテゴリごとに分かれたスプレッドシートへ自動で振り分ける仕組みを、申し込み後7日以内にあなたのGoogleアカウントに組み込んで納品する導入サービス。

顧客はコードではなく「回答が届いた瞬間、必要なシートに自動で並ぶ状態」を買う。

## 想定読者

- 問い合わせや申込をGoogleフォームで受けているが、その後の整理が手作業で残っている人
- 管理者が1〜3人程度の、サークル・研究室・イベント・店舗・個人事業主
- 月の回答件数は数十〜数百件
- ITに強い担当者がいない。でも「毎回コピペ」は限界と感じている
- 一言で言うと「フォームは使えるが、そこから先は気合で乗り切っている人」

## 読者の悩み

- フォームの回答が1枚のシートに溜まるだけで、部門別・担当別に見づらい
- 申込が増えるたびに手動でコピペしてシートを分けている
- 担当ごとに見せたい情報が違うのに、全員が同じ1枚を開いている
- 仕分けミスで「申込を見落とした」「案内が漏れた」が起きる
- シートを分けるたびにヘッダを整える作業でイライラする
- システムを入れるほどではないが、このまま手作業でいくのもつらい

## ターゲット別の響き方

| ターゲット | 刺さる切り口 |
|---|---|
| 大学サークル | イベント申込の部門別・学年別仕分け、引き継ぎが楽になる |
| 研究室 | アンケート・参加申込のカテゴリ別集計準備 |
| イベント運営 | 当日券・前売り・部門別の即時振り分け |
| 小規模事業者 | 問い合わせの種類別仕分け、担当ごとに見せたい情報の整理 |
| 個人事業主 | 複数の申込経路を1枚にまとめず、用途別に分離 |
