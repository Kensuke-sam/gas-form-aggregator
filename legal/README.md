# 法務関連文書

販売開始前に必ず整える。特に匿名運営の場合、これらが揃っていないと信用が毀損する／行政指導の対象となる。

| ファイル | 用途 | LPへの反映 |
|---|---|---|
| `tokutei.md` | 特定商取引法に基づく表記 | footer にリンク必須 |
| `privacy.md` | プライバシーポリシー | footer にリンク必須 |
| `terms.md` | 業務委託規約 | 見積送付時に併せて提示 |

## チェックリスト（公開前）

- [ ] `tokutei.md` の `{}` 箇所をすべて実値に置換
- [ ] 住所・電話は「バーチャルオフィス契約」または「請求時開示」の運用を決定
- [ ] `privacy.md` の連絡先・制定日を記入
- [ ] `terms.md` の屋号・制定日を記入
- [ ] LPにリンクを設置（footer）
- [ ] 問い合わせフォーム冒頭に同意文言を追加
- [ ] これら3つを HTML 化して `docs/legal/` 配下に配置（GitHub Pages で公開）

## HTML化の簡易手順

Markdown → HTML はそのままブラウザで読みにくいため、LPと同じテイストで `docs/legal/tokutei.html` などを用意するとよい。
最低限、`docs/index.html` の CSS を流用して各文書を包めば十分。
