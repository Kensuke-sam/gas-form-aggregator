# 見積書（ひな形）

Googleドキュメントまたは Word で清書してPDFで送付。下記をそのままコピペして使える。

---

```
                     御　見　積　書

{YYYY年MM月DD日}
見積番号：Q-{YYYYMMDD}-{連番}

{お客様名} 御中

下記のとおりお見積もり申し上げます。

─────────────────────────────
件　　　名：  フォーム仕分け自動化パック 導入業務
有効期限 ：  発行日より 30日間
納　　　期：  ご入金確認後 {N}営業日以内
支払条件 ：  銀行振込・前払い
─────────────────────────────

■ お見積もり金額
  合計金額：  ¥{金額}（税込）

■ 内訳

  品目                              数量   単価       金額
  ─────────────────────────────
  {プラン名} プラン                  1     {価格}     {価格}
   - フォーム仕分け機能の組み込み
   - 設定シート（_config / _rules / _log）
   - 運用マニュアル（PDF）
   - 納品後 {30/60} 日の動作サポート
  ─────────────────────────────
  小計                                              ¥{小計}
  消費税（内税）                                    ¥{消費税}
  合計                                              ¥{合計}

■ 作業範囲
  - {具体的な仕分けカテゴリ数・フォーム数}
  - {通知メール設定の有無}
  - {その他の個別要件}

■ 作業範囲外
  - 外部APIとの連携
  - AIによる自動分類
  - フォーム項目の追加・変更後の再設定（別途見積もり）

■ 動作保証・返金
  納品後7日以内に動作確認ができない場合、または受託者の責めに帰す
  べき事由による動作不良の場合、委託料金全額を返金いたします。
  （詳細は業務委託規約 第6条）

■ 振込先
  {銀行名} {支店名}
  {普通/当座} {口座番号}
  口座名義：{名義}

─────────────────────────────
フォーム仕分けラボ
運営担当
{住所（必要に応じて）}
Mail: contact@example.com
─────────────────────────────
```

## Google ドキュメント での作り方

1. Google ドキュメントで新規作成
2. 上記テンプレを貼り付けて整形
3. `{}` を実値に置換
4. ファイル → ダウンロード → PDF
5. メール T3 に添付

## 見積番号の付け方

- 形式：`Q-YYYYMMDD-NN`（例：`Q-20260413-01`）
- 案件IDと紐付けて管理
