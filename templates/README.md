# 顧客送付用テンプレ

見積〜請求〜領収書のひな形。Googleドキュメントでの清書を想定。

| ファイル | 発行タイミング | 備考 |
|---|---|---|
| `quotation.md` | ヒアリング後・入金前 | T3メールに添付 |
| `invoice.md` | 納品時 | 前払いの場合は見積書と兼用可 |
| `receipt.md` | 入金確認後（希望時のみ） | 銀行振込控えで代替可能 |

## 番号体系

| 文書 | プレフィックス | 例 |
|---|---|---|
| 見積書 | Q | Q-20260413-01 |
| 請求書 | INV | INV-20260420-01 |
| 領収書 | R | R-20260425-01 |

案件IDと紐付けて `ops/project-sheet.md` の案件一覧で管理。

## 保管

- PDFを `archives/{YYYY}/{案件ID}/` にまとめて保存（社内）
- 法令に従い税務関連の書類は7年保管
