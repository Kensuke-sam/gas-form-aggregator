# X 運用資料

営業DMの受け皿としてXを使う前提の運用設計。

| ファイル | 内容 |
|---|---|
| `profile.md` | プロフィール文・アイコン・ヘッダー |
| `pinned-post.md` | 固定投稿（スレッド型・単発型） |
| `content-themes.md` | 発信テーマ30案と週次プラン |

## セットアップ順序

1. `profile.md` を参考にアカウント整備
2. `pinned-post.md` 案Aを投稿して固定
3. `content-themes.md` A群を3本投稿して「発信してる感」を作る
4. 営業DMを開始（すでにピンとプロフと3投稿が揃っているので、相手が見ても安っぽくない）

## KPI

初月：
- フォロワー 30〜50
- LP クリック 週5〜10
- DM からの問い合わせ 月 1〜2件

これ以下ならプロフ・固定・投稿内容を見直す。
